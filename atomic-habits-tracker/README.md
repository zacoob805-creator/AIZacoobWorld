## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **Today's Dashboard** | Check off habits, see progress bar, streaks |
| ➕ **Add Habits** | Name, time, duration, habit stacking, Four Laws selector |
| 📅 **30-Day Calendar** | Visual X-mark calendar per habit with streak tracking |
| 🧠 **Identity Builder** | Set your identity statement — who you are becoming |
| 📝 **Weekly Review** | 4-week reflection system based on the book |
| 💾 **Auto-saves** | Everything saves locally in your browser |
| 📱 **Mobile-first** | Works perfectly on phone and desktop |

## 🧠 Based on Atomic Habits by James Clear

This app implements the core frameworks from the book:

- **Identity-based habits** — become the person, not just achieve the goal
- **The Four Laws** — Make it obvious, attractive, easy, satisfying
- **Habit stacking** — attach new habits to existing ones
- **Never miss twice** — the streak calendar keeps you accountable
- **1% better every day** — tracked through streaks and completion rates
